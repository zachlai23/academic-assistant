## UCI Academic Assistant

AI-powered chatbot that helps UCI students create personalized course schedules and graduation plans.

### Features
* **Dynamic Schedule Planning**: Students can receive course plans for a single quarter or a full roadmap until your target graduation date.
* **DegreeWorks Parsing**: Parses DegreeWorks pdfs containing completed and required courses.
* **Personalized Planning**: Custom schedules are provided tailored to user interests and difficulty preferences.
* **AI-Powered Assistant**: Natural conversation interface powered by OpenAI API for context aware academic guidance.

### Tech Stack
**Frontend**: React.js
**Backend**: Python, FastAPI, OpenAI API
**Data**: Anteater API

This project was developed as part of AI @ UCI.
